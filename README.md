# EasyCommerce Desktop

Projeto desenvolvido como **trabalho de conclusão do curso de Delphi** na escola **Tech4Jobs**.

O objetivo do projeto é demonstrar a construção de uma **aplicação desktop em Delphi (VCL)** integrada a banco de dados **PostgreSQL**, aplicando conceitos como:

- CRUD de entidades
- Programação Orientada a Objetos (POO)
- Integração com banco de dados utilizando **FireDAC**
- Relacionamento **Master–Detail**
- Estrutura de aplicação em camadas

---

# 📦 Sobre o Sistema

O **EasyCommerce Desktop** é um sistema simples de **PDV (Ponto de Venda)** que permite:

- Cadastro de produtos
- Registro de vendas
- Inclusão de itens na venda
- Cálculo automático do total da venda
- Persistência de dados no PostgreSQL

O sistema foi desenvolvido com fins **educacionais**, para demonstrar a aplicação prática dos conceitos aprendidos durante o curso.

---

# 🗄️ Banco de Dados

Para utilizar o sistema, é necessário criar as tabelas e inserir dados iniciais no banco de dados.

Execute o script localizado em: \src\utils\CreateDeDadosMockados.sql


Esse script irá:

- Criar as tabelas necessárias
- Inserir dados de exemplo (produtos)
- Preparar o banco para funcionamento do sistema

---

# ⚙️ Tecnologias Utilizadas

- **Delphi (VCL)**
- **PostgreSQL**

---

# 🚀 Execução do Projeto

1. Crie um banco de dados no PostgreSQL.
2. Execute o script:
src/utils/CreateDeDadosMockados.sql


3. Configure a conexão com o banco no projeto Delphi.
4. Compile e execute a aplicação.
